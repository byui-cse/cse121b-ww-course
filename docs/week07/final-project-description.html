Final Project Description

Overview

The final project is an opportunity for you to demonstrate your mastery of the JavaScript language learning outcomes as presented in the course.

Project Requirements

  1. the activities presented in this resource: JavaScript Loops - WDD Learning Modules
  2. Given the following code initializations: Code Works The page loads and runs (using JavaScript code!) Organization: Multiple Functions Show some level of organization: at a minimum it should have more than one function. DOM Interaction Include DOM interaction: select an element, modify it, and listen for and react to events. Conditional Branching Show examples of conditional branching. Use conce from course. At least one of the following should be used: Fetch Objects Modules Arrays and array method usage (forEach, map, reduce, filter, etc) Video Video showing student's face and source code review.
