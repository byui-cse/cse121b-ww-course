CSE 121B: JavaScript Language

W02: Programming Task

Overview

In this assignment you are going to replace your cse121b site home page with a biography page with a menu. The responsive menu will be provided to you. The task is to complete the required steps and supplement the content of the page using JavaScript.

Remember, that in order for us to talk to the HTML document, the HTML must reference the source of the JavaScript so that the HTML DOM can be manipulated (edited). This JavaScript file(s) will be external to the HTML document. The <script> element will be used to reference the JS file.

Instructions

Setup

  1. Download the starter files and place them in the root directory of your cse121b repository.
  2. In VS Code, delete the original rename the .html file to index.html
  3. Note the folders which follow a common naming convention for the web.

Variables

  1. Declare and instantiate a variable to hold your name. Identify the variable as "fullName".
  2. Declare and instantiate a variable to hold the current year. Identify the variable as "currentYear".
  3. Declare and instantiate a variable to hold the file path (location) and file name of a profile picture of yourself.

Manipulating the Document

  1. Use the document.getElementById() method to get the element with the id of "fullName".
  2. Use the innerHTML property to set the value of the element to the value of the variable "fullName".
  3. Use the document.querySelector() method to get the element with the id of "currentYear".
  4. Use the textContent property to set the value of the element to the value of the variable "currentYear".
  5. Use the document.getElementById() method to get the img element with the id of "profilePicture".
  6. Use the src property to set the value of the element to the value of the variable "profilePicture".
    Hint Use document.querySelector().setAttribute()

Submission

  1. Commit and push your changes to GitHub.
  2. Submit the URL of your GitHub repository to I-Learn.
