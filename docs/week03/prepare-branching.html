JavaScript Conditions and Branching

Overview

There are common control structures in all programming languages that allow you to control the flow of the program including sequence, branching, looping, and functions. In JavaScript, these decision blocks are the if, else, else if, switch, and case statements. These statements allow you to control the flow of the program based on the Boolean result of a condition.
